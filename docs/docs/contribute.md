# Contribute

You can contribute to the development and improvement of the Food You app in several ways.

!!! danger "Code contributions not accepted"
    Code contributions are currently not accepted due to licensing restrictions and the fact that this
    app generates some income.

## Reporting Issues

If you encounter any bugs or issues while using the app, please report them
on [GitHub](https://github.com/maksimowiczm/FoodYou/issues). When reporting an issue, please use the
provided template to ensure that all necessary information is included. This will help address the
issue more efficiently.

## Feature Requests

You can participate in discussions about new features or suggest your own ideas
on [GitHub](https://github.com/maksimowiczm/FoodYou/issues). When suggesting a new feature, please
provide as much detail as possible about the functionality and its potential benefits.

!!! warning
    Please make sure to check existing issues and feature requests before submitting a new one to
    avoid duplicates.

## Translations

Contributions to translate the app into other languages are welcome. Help improve international
accessibility via [Crowdin](https://crowdin.com/project/food-you). If you want to be credited for
your translation work,
please [message me on Crowdin](https://crowdin.com/messages/create/16826711/767947).
